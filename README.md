README - موقع بسيط لشركة تليكوم للأقمار الصناعية

ملفات موجودة:
- index.html   -> صفحة الموقع الرئيسية (RTL، عربية)
- styles.css   -> تنسيقات التصميم
- logo.png     -> استبدلي هذا الملف بلوجو شركتك (حالياً مجرد عنصر نوري placeholder)
- README.txt   -> هذا الملف

خطوات التشغيل المحلية:
1) حمّلي الملفات على حاسوبك.
2) افتحي index.html في المتصفح (نقرة مزدوجة).
3) لتشغيل الفورم: سجّلي في https://formspree.io/ وأنشأي فورم جديد، ثم استبدلي قيمة attribute 'action' في الفورم بعنوان Formspree الخاص بيك.

نشر مجاني (سهل):
- Google Sites: أنسب للمبتدئين (سهل، سحب وإفلات) — تحتاجي حساب Google.
- GitHub Pages: لنشر مجاني من ملفات HTML (لو عندك حساب GitHub).
- Netlify / Vercel: نشر مجاني ودعم للنماذج والوظائف الإضافية.

لو حابة أرفع الموقع على استضافات مجانية (GitHub Pages أو Netlify) أقدر أوجّهك خطوة بخطوة أو أعمللك الخطوات لو اعطيتيني صلاحية مؤقتة (أو تديلي حساب مؤقت).
